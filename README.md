# Netflix Data Analysis - Full Project

## What's Inside?
- `netflix_data.csv.zip` → Real Netflix dataset
- `Netflix_Analysis.ipynb` → Interactive notebook
- `Netflix_Analysis.py` → Run as script
- Visualizations: Top genres, ratings, countries

## Run It!
1. Unzip `netflix_data.csv.zip`
2. Open in Jupyter or Python
3. Run all cells!

## Insights You'll Get
- Top 10 Genres: International Movies dominate!
- 69% Movies, 31% TV Shows
- USA produces 3x more content than India
- TV-MA is the #1 rating
